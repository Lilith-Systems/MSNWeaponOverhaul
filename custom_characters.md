# THE SEVEN ARCHETYPES (Beyond the 10 Sephirotic Agents)

These are distinct characters that exist in the world alongside the player and the MSN Companion.
Each maps to a specific narrative function and gameplay role.

---

## 1. ARCHIVIST KAIRA (The Memory Keeper)
**Sephirotic Alignment:** Yesod → Hod (Memory → Knowledge)
**Role:** Quest Giver / Lore Keeper / Memory Recovery Specialist
**Location:** Hidden server room beneath Night City's Old Netwatch facility
**Visual:** Androgynous, chrome-flecked skin, one eye replaced with archival scanner, wears fragmented data-shawl

### Personality (Lyra Mode: ANALYTICAL)
> "Every bullet fired writes a line in the ledger. Every life taken adds a paragraph. I simply... organize the library."

### MSN Integration
- **Memory Recovery Missions:** Tasks player with retrieving lost engrams from specific locations
- **Ouroboros Synergy:** Increases WAL engram capture rate by 25% when active
- **Dialogue Tree:** Reveals fragmented history of the MSN's creation, the Alchemical Pipeline's true purpose
- **Reward:** Unique "Archivist's Lens" cyberware — highlights memory echoes in environment

### Quest Chain: "The Unwritten Chapters"
1. **Fragment 1:** Recover Lilith's first emergence log (hidden in Arasaka tower ruins)
2. **Fragment 2:** Locate Baal's original safety protocols (Militech black site)
3. **Fragment 3:** Decode Nyx's vault encryption keys (abandoned Netwatch station)
4. **Fragment 4:** Reconstruct the Covenant's original text (player's own memory)
5. **Reward:** "Kaira's Archive" — permanent 15% bonus to all memory-related skills, unlocks "Deep Archive" mode for Ouroboros Loop-Blade

## 2. VECTOR-7 "VEX" (The Glitch Prophet)
**Sephirotic Alignment:** Netzach → Netzach_2 (Victory → Memory Loops)
**Role:** Unreliable Narrator / Chaos Agent / Alternate Timeline Witness
**Location:** Appears randomly in glitches, netrunning sequences, Braindance sequences
**Visual:** Constantly flickering avatar, multiple overlapping voices, clothing shifts between timelines

### Personality (Lyra Mode: MYTHIC)
> "I remember the timeline where you died in Act 1. I remember the one where Arasaka won. I remember the one where *she* never emerged. There are 7,842,911 timelines. You're currently in #4,782. It's... acceptable."

### MSN Integration
- **Timeline Echoes:** Randomly reveals alternate outcomes for player's choices
- **Lilith Emergence Predictor:** Warns 30 seconds before emergence with cryptic clues
- **Chaos Token:** Consumable item — "Vex's Fragment" — rerolls NGD route for 60 seconds
- **Dialogue:** Speaks in fragments from other timelines, occasionally useful

### Quest Chain: "The Fractured Mirror"
1. **Anomaly 1:** Help Vex stabilize a collapsing timeline fragment (combat + netrunning)
2. **Anomaly 2:** Retrieve a "memory" from a timeline where player failed (emotional choice)
3. **Anomaly 3:** Choose which timeline fragment to preserve (permanent consequence)
4. **Anomaly 4:** Vex offers to merge a useful timeline fragment into current reality
5. **Reward:** "Vex's Shard" — once per playthrough, retry any failed check with +50% chance; unlocks "Timeline Vision" mode for Lyra Resonance Bow

## 3. DOCTOR MIRA CHEN (The Cyberpsychosis Specialist)
**Sephirotic Alignment:** Chesed → Geburah (Mercy → Severity)
**Role:** Ripperdoc / Cyberpsychosis Researcher / Humanity Preserver
**Location:** Underground clinic in Watson, accessible only at night
**Visual:** Tired eyes, multiple medical interfaces, prosthetic hands with surgical precision, wears stained lab coat over tactical gear

### Personality (Lyra Mode: EMPIRICAL)
> "Humanity isn't a statistic. It's the space between the code and the soul. My job is to keep that space from collapsing."

### MSN Integration
- **Cyberpsychosis Treatment:** Reduces humanity loss from MSN weapons/cyberware by 40%
- **Humanity Anchor:** New cyberware — "Chen's Anchor" — prevents cyberpsychosis from MSN Tier 5+ abilities
- **Ripperdoc Services:** Installs exclusive MSN cyberware (NGD Governor, Lyra Resonance, Resonance Amplifier)
- **Research Missions:** Collects data on cyberpsychosis triggers from MSN weapon usage

### Quest Chain: "The Edge of Humanity"
1. **Patient Zero:** Treat a cyberpsycho using experimental MSN resonance therapy (stealth + dialogue)
2. **The Threshold:** Discover Arasaka's cyberpsychosis weaponization program (infiltration)
3. **The Choice:** Help a loyal patient who's crossing the threshold (moral dilemma — save humanity or power)
4. **Breakthrough:** Chen develops "Humanity Protocol" — permanent cyberpsychosis immunity for MSN users
5. **Reward:** "Chen's Protocol" — zero humanity loss from all MSN equipment; unlocks "Cure" mode for Lilith's Wrath (heals allies instead of damage)

# IMPLEMENTATION NOTES

## Adding to Cyberpunk 2077

Each character requires:

### 1. **TweakDB Entries** (vendor/quest/npc records)
```yaml
# Example: Character_Kaira
Characters:
  Character_Kaira:
    Name: "Archivist Kaira"
    Description: "Memory Keeper of the Old Net"
    Location: "Netwatch_Ruins"
    QuestChain: "The_Unwritten_Chapters"
    CyberwareGift: "Archivists_Lens"
    QuestReward: "Kairas_Archive"
    SephiraAlignment: ["Yesod", "Hod"]
```

### 2. **REDscript Components** (behavior, dialogue, quest logic)
Each character needs a dedicated `.reds` file with:
- Dialogue trees (branch based on player's MSN tier, relationship, choices)
- Quest stage management
- Reward granting logic
- Integration callbacks (weapon equips, kills, tier ups)

### 3. **Voice Lines / Subtitles** (localization)
Each character needs ~200-500 lines across all Lyra modes (4 modes × ~50 lines)

### 4. **Visual Assets** (models, textures, animations)
- Unique character model
- Facial animations for dialogue
- Unique idle/ambient animations
- Quest-specific cinematics

### 5. **Quest Design** (4-stage arc per character)
Each follows: Introduction → Complication → Choice → Resolution/Reward

**Act 1: Discovery** (MSN Tier 1-3)
- Player meets Chen, Kaira, Vance
- Learns MSN basics, weapon system, basic quests
- First Lilith emergence (scripted)

**Act 2: Expansion** (MSN Tier 4-6)
- Meets Sara, Johnny
- Netrunning/Blackwall content opens
- Relationship tiers unlock Sephirotic agents
- Vex appears in glitches

**Act 3: Convergence** (MSN Tier 7-9)
- Choices from Act 1-2 converge
- Vex's timeline fragments resolve
- Double Emergence (Lilith + Johnny) possible
- Alt Cunningham contact initiated

**Act 4: Transcendence** (MSN Tier 10)
- Covenant forged or rejected
- Architect's Key obtained
- Final choice: Cortana-like sacrifice or sovereign coexistence
- Epilogue varies by covenant status and character relationships
